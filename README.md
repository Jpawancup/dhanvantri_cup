# Dhanvantri Healthcare Ecosystem

Welcome to the Dhanvantri monorepo.

## Project Structure
- `frontend-web/`: Web application built with Next.js 14, TailwindCSS, ShadCN UI.
- `mobile-app/`: Mobile application built with React Native (Expo). (Upcoming)
- `backend/`: Backend server built with NestJS, PostgreSQL. (Upcoming)
- `docs/`: Documentation and project specifications.

## Prerequisites
- Node.js >= 18.x
- npm >= 9.x

## Phase 1
Phase 1 focuses exclusively on the Complete UI design, including all pages and components needed for the Healthcare and Social portions of the app.

For more detailed architectural overview, consult `dhanvantri_project.md`.
